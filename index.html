    About Trivia Challenge Pro

    This is a lightweight, interactive quiz application designed for practice and fun. It features multiple-choice questions fetched live from the Open Trivia Database.

    Features:

    • 15-second timer per question for a fast-paced challenge.
    • Instant scoring and feedback.
    • A persistent leaderboard to track top scores.
